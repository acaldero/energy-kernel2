## Basic pre-requisites
```console
apt-get -y install build-essential
apt-get -y install gnuplot gnuplot-data gnuplot-doc
pip3 install matplotlib
```

## Run
```console
mkdir -p results
./clean.sh
./compile.sh
./execute.sh
./plot.sh
```
